# 1. プロトコルの改善ポイント

## 現在のプロトコル

1. サンプルを2 µg/10 µL in PBSとなるようにプレートに添加
2. PNGase Fを0.5 µL添加し、室温で一晩静置
   - CetuximabのようにFabにも糖鎖がつくものはRapid PNGase Fで処理
3. O型糖鎖処理をする場合は、OglyZORとSialEXOを0.5 µLずつ添加し、室温で一晩静置
4. 還元処理するサンプルには50 mM TCEPを2 µL添加し、37℃で2時間以上静置
5. サンプルにギ酸アンモニウムを40 µL添加し、混和する
6. Orbitrapで測定

## 現行法で最初に明確化すべき点

### 1. 酵素量を体積だけで管理している

「PNGase F 0.5 µL」では、製品を変更したときに酵素量を比較できない。通常型PNGase FとRapid PNGase Fでは活性単位の定義も異なり、Rapid型は「1 reaction/µL」で販売されている。

最低限、以下を実験記録に残す。

- メーカー
- 製品番号
- ロット番号
- 酵素濃度または活性（U/µL）
- 添加した総活性（U）
- 基質1 µg当たりの活性（U/µg）
- 最終反応液量
- バッファーの種類と最終濃度

### 2. 室温overnightは通常型PNGase Fの標準条件ではない

通常型PNGase Fの推奨条件は一般に37℃である。NEBの通常型PNGase Fでは、23℃の活性は37℃より低く、Native基質ではより多くの酵素と長い反応時間が必要とされている。

室温overnightを現行基準として残しつつ、少なくとも以下と比較する。

- 37℃・4時間
- 37℃・16時間
- 45℃・1時間
- 50℃・10分、30分、60分

### 3. Rapid PNGase Fを室温overnightで使わない

Rapid PNGase Fは専用バッファーと加熱を含む反応システムである。標準的には50℃・10分で反応させる。Fab N型糖鎖など切れにくい部位では、抗体とRapid Bufferだけを80℃・2分加熱し、冷却してから酵素を添加し、50℃・10分反応させる。

Rapid型を通常型と同様に室温overnightで使用しても、Rapid型の本来の性能比較にはならない。

### 4. TCEP濃度の表現が曖昧

「50 mM TCEPを2 µL添加」が50 mMストックを2 µL加えるという意味なら、最終濃度は50 mMではない。

- N型処理のみ：10 µL試料＋0.5 µL酵素＋2 µL TCEPの場合、50 mMストックの最終濃度は約8 mM
- N型＋O型処理：総液量がさらに増えるため、最終濃度は約7～8 mM

したがって、記録は以下のいずれかに統一する。

- 「50 mM TCEP stockを2 µL添加」
- 「最終濃度50 mMとなるようにTCEPを添加」

還元条件の検討では、最終濃度として例えば5、10、25、50 mMを比較する。反応時間も30分、1時間、2時間で評価する。

### 5. ギ酸アンモニウム添加は脱塩ではない

PBS試料にギ酸アンモニウムを加える操作はPBSの希釈であり、NaClやリン酸を除去するバッファー交換ではない。反応バッファー、酵素保存液、還元剤も残るため、次の問題が起こり得る。

- イオン化抑制
- Na/Kアダクト
- ピーク幅の増大
- チャージ分布の変化
- サンプル間での塩濃度差

通常型、Rapid型、O型酵素を公平に比較する場合は、反応後の全試料を同じ方法で脱塩またはバッファー交換する。候補はmicrodialysis、限外ろ過、spin desalting columnなどである。

### 6. プレートでのovernight反応は蒸発の影響を受ける

10 µL程度の小容量を室温または37℃で長時間保持すると、ウェル位置やシール状態による蒸発差が反応濃度差になる。

- PCRプレートまたはlow-bindチューブを使用する
- 加熱対応シールを使用する
- 反応前後の液量を確認する
- edge wellと中央wellの差を確認する
- 可能ならthermocyclerのheated lidを使用する

## 検討できる因子

| 因子 | 推奨する検討範囲 | 主に分かること |
|---|---|---|
| 酵素の種類 | PNGase F、Rapid P0710、Rapid P0711、EndoS/EndoS2、Endo H、PNGase A | 基質特異性、Native構造への依存性 |
| 酵素量 | 0.25×、1×、4×、またはU/µg系列 | 酵素律速か基質アクセス律速か |
| 温度 | 室温、37、45、50℃、必要時60～80℃前処理 | 反応速度と熱変性のバランス |
| 時間 | 10分、30分、1、4、16時間 | 反応完結時間、長時間処理由来アーティファクト |
| pH | 6.0、6.8、7.5、8.0を製品適合範囲で設定 | 活性、基質構造、凝集への影響 |
| 還元条件 | なし、TCEP 5～50 mM | ジスルフィド依存の糖鎖アクセス性 |
| 前処理 | なし、熱、還元、界面活性剤 | Fab/Fc糖鎖の露出への寄与 |
| 反応順序 | N→O、O→N、同時処理 | 酵素間のバッファー・温度適合性 |
| バッファー | PBS、メーカー推奨、MS-compatible buffer | 活性とMSスペクトル品質 |
| 混和 | 静置、軽い振とう | 拡散律速、再現性 |

## 推奨する段階的な検討

### Phase 1：現行条件の再現性確認

同一サンプル、同一ロットで最低3反復行い、以下を確認する。

- 脱糖鎖率
- 回収率
- deconvolution後の主ピーク質量
- 未消化体および部分消化体の割合
- ピーク幅とS/N
- プレート内位置による差

### Phase 2：N型糖鎖処理のスクリーニング

| 条件 | 酵素 | 反応条件 | 目的 |
|---|---|---|---|
| N0 | なし | 現行操作 | 未処理対照 |
| N1 | 通常型PNGase F | 室温・overnight | 現行基準 |
| N2 | 通常型PNGase F | 37℃・4時間 | 標準温度での短縮 |
| N3 | 通常型PNGase F | 37℃・16時間 | 標準温度での完全性 |
| N4 | 通常型PNGase F | 45℃・1時間 | 加熱による高速化 |
| N5 | Rapid P0710 | 50℃・10分 | Rapid one-step |
| N6 | Rapid P0710 | 80℃・2分前処理→50℃・10分 | Fab糖鎖へのアクセス |
| N7 | Rapid P0711 | 75℃・5分前処理→50℃・10分 | 非還元形式 |
| N8 | EndoS2 | メーカー推奨Native条件 | Fc特異的な穏やかな処理 |

基質として、次の2種類を並べると解釈しやすい。

- Fc糖鎖のみを持つ一般的なIgG
- FcとFabの両方に糖鎖を持つCetuximab

### Phase 3：O型糖鎖処理のスクリーニング

| 条件 | 酵素 | 反応条件 | 目的 |
|---|---|---|---|
| O0 | なし | 同一バッファー | 未処理対照 |
| O1 | SialEXOのみ | 37℃ | シアル酸由来質量不均一性の確認 |
| O2 | OglyZORのみ | 37℃ | シアル酸依存性の確認 |
| O3 | OglyZOR＋SialEXO | 37℃・1～2時間 | 現行酵素系の標準条件 |
| O4 | NEB O-Glycosidase＋neuraminidase | 37℃・1～4時間 | 別由来酵素との比較 |
| O5 | O-Glycoprotease IMPa | 37℃・2時間 | O糖鎖部位依存切断の確認 |

OglyZORやNEB O-Glycosidaseは、O型糖鎖の全構造を無条件に除去する酵素ではない。特にcore 1/core 3以外、伸長型、分岐型、修飾型では残存する可能性がある。

### Phase 4：温度・酵素量の最適化

Phase 2と3で有望だった酵素について、次の小規模マトリクスを組む。

- 酵素量：0.25×、1×、4×
- 温度：37、45、50℃
- 時間：10分、1時間、4時間

すべてを総当たりする必要はない。まず低量・標準量、高温短時間・標準温度長時間を対比し、反応が酵素量依存か、温度依存かを確認する。

## N型とO型の反応順序

SialEXOとOglyZORはNative条件で同時反応可能である。一方、Rapid PNGase Fの80℃前処理を同じ反応液中で行うと、O型酵素を失活させる可能性がある。

両方を処理する場合は、以下の順序が扱いやすい。

1. SialEXO＋OglyZORをメーカー推奨Native条件で反応
2. 必要に応じてバッファー調整
3. Rapid PNGase Fの前処理および反応
4. 全試料を同じ方法で脱塩またはバッファー交換
5. 還元群だけTCEP処理
6. Orbitrap測定

## 必須コントロール

- 未処理サンプル
- 酵素なし、バッファーのみ
- 酵素なし、加熱のみ
- 酵素のみのブランク
- 完全変性・完全脱糖鎖の陽性対照
- 還元剤なし、還元剤あり
- 同一条件のtechnical replicate

## intact MSで比較する評価指標

- 理論質量との差
- 完全脱糖鎖体の割合
- 片側のみ脱糖鎖された中間体
- Fcは処理されたがFab糖鎖が残った分子種
- H鎖・L鎖の回収率
- 新規fragment
- 高分子量体、凝集、沈殿によるシグナル低下
- チャージ分布
- ピーク幅
- Na/K/バッファーアダクト
- 反応前後の総LCピーク面積
